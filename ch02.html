数学の道具箱② 内積と cosθ

2つの方向がどれだけ揃っているかを計る「内積」。CGで最も頻出の演算です。

角度と cos の直感

数学の教科書では cos（コサイン）は「複雑な関数」に見えるかもしれません。 でも CGでは、cos は「2つのベクトルがどれだけ揃っているか」を測る道具なのです。

例えば、懐中電灯を想像してください：

  • 壁に垂直に当てると → 光が強い ← cos(0°) = 1
  • 45°斜めに当てると → 光が弱くなる ← cos(45°) ≈ 0.707
  • 90°（壁に平行）に当てると → 光が当たらない ← cos(90°) = 0
  • その先（逆向き）に当てると → 負の値 ← cos(120°) = −0.5

つまり、cos(θ) は「0 から 1 の範囲」で「揃い具合」を表すのです。

cos のグラフ

0° から 180° までの cos を見ると、この関係が一目瞭然です：

横軸：角度（度） | 縦軸：cos値（−1 ～ 1）

内積（ドット積）の定義

2つのベクトル a = (a₁, a₂, a₃) と b = (b₁, b₂, b₃) があるとき、 内積は以下のように計算します：

a · b = a₁·b₁ + a₂·b₂ + a₃·b₃
意味：対応する成分どうしを掛けて、全部足す

例：a = (1, 2, 3)，b = (4, 5, 6) なら：

a · b = 1×4 + 2×5 + 3×6 = 4 + 10 + 18 = 32

内積と角度の関係

内積には素晴らしい性質があります。2つのベクトルがともに正規化されている（長さ1）なら：

normalized(a) · normalized(b) = cos(θ)
意味：正規化されたベクトルの内積は、その間の角の cos になる

この性質がCGで最強な理由：

  • 「2つの方向がどれだけ揃っているか」を1つの数で表せる
  • その数は −1 から 1 の範囲
  • 計算が超高速（成分の掛け算と足し算だけ）

W2-a: 内積と角度の対話ツール

2本の矢印をドラッグして、角度を変えてみてください。 右側に「角度」「cos値」「内積値」が同時に表示されます。

ベクトルを長くしたり短くしたり、向きを変えたりできます。

角度: 0°
cos値: 1.000
内積値: 0.000
a·b長さ²: 0.000

CGで最重要な式：max(0, n·l)

ライトが当たっているかどうかを判定するのに、これを使います：

brightness = max(0, n · l)
意味：法線 n とライト方向 l がどれだけ揃っているか（0 以下なら 0）

この式の意味：

  • ライトが表面に垂直に当たる（n と l が同じ向き） → n·l = 1 → 明るい
  • 斜めから当たる → n·l = 0.5 など（0 ～ 1） → やや暗い
  • ライトが裏側から当たる（n と l が逆向き） → n·l < 0 → max(0, ...) = 0 → 暗い（完全に黒）

max(0, ...) は「0 以下なら 0、以上ならそのまま」という意味です。 つまり、「裏側は絶対に照らされない」という現実を反映しています。

W2-b: Lambert シェーディング

球体の各点で n·l を計算して、色で可視化したものです。 ライト方向を動かすと、どの向きの面が「揃い具合」で明るくなるか、一目瞭然です。

白い部分：ライトが垂直に当たっている | グレー：斜めから当たっている | 黒い部分：裏側（当たっていない）

内積の3つの計算方法

内積には異なる3つの見方があります。全部同じ値を返しますが、直感が違います：

①成分の掛け算の和

a · b = a₁b₁ + a₂b₂ + a₃b₃
コンピュータが実際に計算する形

②大きさと角度の関係

a · b = |a| · |b| · cos(θ)
2つのベクトルの「揃い具合」を見る見方

③投影（影）の考え方

a を光源とし、b に垂直に光を当てたとき、 a の「影の長さ」が内積。

a · b = |b| × （a を b の方向に投影した長さ）
ライティングでは、この「影」の概念が大事

実装形式（GLSL の例）

シェーダで n·l を計算するコード：

float brightness = max(0.0, dot(normal, lightDir));
// dot = 内積の関数。GLSL では dot(a, b) で a·b を計算

まとめ

  • 内積は「2つのベクトルが揃っているか」を測る演算
  • 正規化されたベクトルの内積は cos(θ) に等しい（−1 ～ 1）
  • max(0, n·l) は、光が当たっているかどうかを判定する最基本の式
  • 内積は CGで超頻出。速く、直感的で、強力
