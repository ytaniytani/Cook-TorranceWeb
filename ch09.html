スペキュラ成分③: F項（フレネル / Schlick近似）

「浅い角度で見ると、何でも鏡になる」——身の回りで毎日起きている物理現象が、metallic パラメータの正体につながります。

まず現象から：水面と机

湖を思い出してください。足元の水面は底が透けて見えますが、 遠くの水面は空を映す鏡になっています。同じ水なのに、です。

机でも同じ実験ができます。目の高さを机の面すれすれまで下げて遠くを見ると、 普段は反射なんてしないつや消しの机が、窓や照明をうっすら映し出します。

フレネル効果：面を正面から見ると反射は弱く、 浅い角度（すれすれ）で見るほど反射が強くなり、90°の極限では何でも100%の鏡になる。 これはあらゆる物質に共通の物理法則です。

式（Schlick 近似）

FSchlick = F0 + (1 − F0) · (1 − v·h)⁵
意味：正面では F0、すれすれ（v·h→0）では 1.0 に向かって急カーブで立ち上がる
  • F0 — 正面（垂直入射）から見たときの反射率。材質固有の値
  • v·h — 視線とハーフベクトルの揃い具合。正面で1、すれすれで0
  • 5乗 — 「端で急に立ち上がるカーブ」を作るための近似。 物理の厳密式（フレネル方程式）をゲーム向けに軽くしたもので、精度は十分

W9-a: フレネルカーブ

横軸は視線の角度（0°=正面、90°=すれすれ）、縦軸は反射率。 F0 を変えても、90° 側では必ず 1.0 に到達することを確認してください。

metallic パラメータの正体はここ

Substance や UE の metallic スライダーは、実はこの F0 の切り替えスイッチです：

F0 = mix(0.04, baseColor, metallic)
metallic=0 → F0は0.04（非金属）／ metallic=1 → F0はベースカラーそのもの（金属）
  • metallic = 0（非金属）：F0=0.04 の無色の反射＋ベースカラーは拡散（物体の色）に使われる
  • metallic = 1（金属）：ベースカラーが F0 になり反射自体に色がつく。 拡散は消滅する（金属内部に光が潜れないため。第11章で詳しく）

W9-b: 球体で F 項だけを見る

F 項だけを表示した球体です。縁（りん郭）に近いほど明るくなる「リム」が見えます。 球の縁は視線に対してすれすれの角度＝フレネルが強く効く場所だからです。

まとめ

  • フレネル効果：浅い角度ほど反射が強い。90°では何でも100%の鏡
  • Schlick 近似の5乗は「端で急に立ち上がるカーブ」を作る道具
  • F0 は正面反射率。非金属は一律 0.04、金属はベースカラーそのもの
  • metallic スライダーの正体は F0 と拡散の切り替え
