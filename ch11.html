第11章

全体の合成とエネルギー保存

D・F・G が出揃いました。すべての部品を1本の式につなぎ、スライダーから最終RGBまでを電卓レベルで完全トレースします。

完成形の式

f = kd · (albedo / π) + D·F·G / (4·(n·l)·(n·v))
kd = (1 − F) · (1 − metallic)
Lo = f · Lc · (n·l)
意味：拡散＋鏡面を kd で配分し、光の色と斜め補正を掛けて画面の明るさに

kd の意味 — スペキュラに使った光は、拡散に回らない

表面に届いた光の使い道は2つに1つです。 表面で跳ね返る（スペキュラ）か、内部に潜って散乱する（ディフューズ）か。 同じ光を両方に使うことはできません。

スペキュラに使われた割合はフレネルの F そのものです。 だから拡散に回れるのは残りの (1 − F)。 これがエネルギー保存の会計係 kd の正体です。

(1 − metallic) はなぜ掛かる？ 金属の内部には自由電子が詰まっていて、潜り込んだ光は即座に吸収されます。 つまり金属では拡散が発生しません。metallic=1 で拡散をゼロにするための係数です。

metallic ワークフローまとめ

第9章と合わせると、metallic スライダー1本の裏で起きていることはこの2つです：

  • F0 = mix(0.04, albedo, metallic) — 反射率と反射の色が切り替わる
  • kd = (1−F)·(1−metallic) — metallic=1 で拡散が消滅する

metallic=1 のとき、albedo は「物体の色」ではなく「反射の色（F0）」として使われる—— これが金属ワークフローの本質です。

試してほしい実験

  • ハイライトの条件：ライトと視線の角度を同じにすると h が n と一致して n·h=1 → D が最大に
  • metallic を 1 へ：kd → 0 になり diffuse の行が消滅、F0 がベースカラーになる様子を追う
  • すれすれ実験：視線の角度を 85° に → (1−v·h)⁵ が跳ね上がり F が 1 に近づく（第9章）、 同時に G1(n·v) が下がる（第10章）——2つの項の綱引きを数値で見る
  • roughness の両端：0.05 と 1.0 で D のピーク値の桁の違いを見る（第8章）

まとめ

  • f = kd·(albedo/π) + D·F·G/(4(n·l)(n·v))。役者は全員登場済み
  • kd = (1−F)(1−metallic)。スペキュラに使った光は拡散に回らない（エネルギー保存）
  • metallic は「F0の切り替え」と「拡散の消滅」の2つを同時に操作するスイッチ
  • スライダーから最終RGBまで、全ステップが四則演算で追えるようになった——完全理解まであと1章
